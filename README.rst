
==================
Leonardo redirects
==================

Allows to set up redirects based on django.contrib.redirects.

.. contents::
    :local:

Installation
------------

.. code-block:: bash

    pip install -e git+https://github.com/leonardo-modules/leonardo-module-redirects#egg=leonardo_module_redirects

    python manage.py migrate
